## 👋 About This Portfolio

This is my personal developer portfolio built with **React**, **TailwindCSS**, and **Vite**. It showcases my projects, technical skills, and background as a Computer Science graduate with a focus on backend engineering and full-stack development.

---

## ✨ Features

- 🌑 **Light/Dark Mode** with local storage persistence
- 💫 **Animated UI** — stars, meteors, scroll effects, glowing elements
- 📱 **Fully Responsive** — mobile, tablet, and desktop
- 🖼️ **Projects Showcase** — live demos, GitHub links, tech stack tags
- 📊 **Skills Section** — categorized tech stack display
- 📩 **Contact Section** — social links and contact form

## 🚀 Run Locally

```bash
git clone https://github.com/SaitejaBandari756/Portfolio
cd Portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

---

## ☁️ Deployment

Deployed as a static site on **Render**:

1. Push to GitHub
2. Render → New → Static Site
3. Build Command: `npm install && npm run build`
4. Publish Directory: `dist`
